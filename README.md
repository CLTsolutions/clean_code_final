# Getting Started

Getting the code

```bash
git clone git@github.com:CLTsolutions/clean_code_final.git
```

After cloning change terminal location
into the folder

```bash
cd clean_code_final
```

Installing Needed Node modules so the app
can run

```bash
npm install
```

Opening your editor

```bash
code .
```

Starting the server

```bash
npm start
```
